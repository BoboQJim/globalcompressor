Got Questions?

Frequently Asked Questions

Everything you need to know about using Squoosh.online.

How does Squoosh.online compress files without uploading them?
Squoosh.online uses WebAssembly (WASM) technology to run powerful compression algorithms directly inside your web browser.

What image formats does Squoosh.online support?
Our image tools support all major web image formats including JPEG, PNG, WebP, AVIF, GIF, SVG, and BMP.

Is there a file size limit for compression?
The file size limit depends on your device's available memory. Most devices handle files up to several hundred MB.

Do I need to create an account to use the tools?
No. Squoosh.online is completely free and requires no registration or personal information.

Is my data safe when using Squoosh.online?
Absolutely. Files are processed entirely within your browser. No data is transmitted to any server.
